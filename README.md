# 极客时间 Go 初级工程师训练营

这是为极客时间 Go 初级工程师训练营提供的配套代码仓库。

整个仓库的代码可以分成两个部分：
- 基础语法部分：在 syntax 中。
- 小微书代码。

## 如何使用该仓库？

因为仓库中的代码是随着课程推进而演进的，所以你需要通过 git log 来查看提交记录。

其中：
- master 分支是我预先写好的代码
